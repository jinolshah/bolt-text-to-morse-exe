Text to Morse Code using Bolt IoT

  • Post author By Jinol
  • Post date March 26, 2021
  • No Comments on Text to Morse Code using Bolt IoT

After being recruited by Kingsman, I soon grew weary of converting my messages to Morse code every time I wanted to communicate with other secret agents. As a result, I decided to make a super secret application (feel free to download it from here) to automate my process using the Bolt IoT module.

Things used in this project

Hardware Components

  • Bolt IoT Module
  • Buzzer
  • LED
  • Breadboard
  • Jumper Cables
  • MicroUSB Cable

Software, Apps and online services

  • Download direct application (windows10):
    • Bolt Integrated Text To Morse Converter
  • Or to build it yourself:
    • Python
    • Visual Studio Code (or any other code editor of your choice)

Hardware Setup

  1. Bolt Module Connections
    • Positive pin of Buzzer to pin “0” of Bolt module and negative pin of Buzzer to pin “GND” of Bolt module.
    • Positive pin of LED to pin “1” of Bolt module and negative pin of LED to pin “GND” of Bolt module.
    • A breadboard can be used for these connections.

Software Setup

  • Using my application directly
    1. Download the Bolt Integrated Text To Morse Converter from here.
    2. Run “Bolt Text To Morse.exe” and install it at your preferred location.
    3. From the installed folder run “TextToMorse.exe”. It will open up a command prompt window and a new tab of the web app in you default browser.
    4. Do not close the command prompt window while using the application.
    5. Once you’ve setup your hardware and made the required connections, enter the required details in the application and press “Convert” to convert the character you entered in to Morse code.
    6. You’ll find your device ID here and the API Key here.
  • Installations for building the application yourself
    1. Install Python on your device.
    2. Install VSCode or any other code editor on your device.
    3. Type CMD in your search and press enter to open up command prompt.
    4. Install the following modules.
  • Writing the code
    1. Make a directory for your project.
    2. You can get the source code for the application from this GitHub repository.
    3. You can refer to the code and write the program yourself or clone the entire repository.
  • Understanding the code
    1. “app.py” is the main bit of the code. It executes the python program as a web-app.
    2. It imports flask, boltiot, webbrowser modules and two other python scripts.
    3. “forms.py” defines different variables which take input from the user. Also, these variables have validators which makes sure that the user inputs valid data. You can also provide the message to flash in case the data entered is invalid.
    4. “textmorse.py” consists of a function which converts the input character to Morse and returns it as a string. Originally the function was meant to be used for a string of characters but later it was limited to a single character due to Bolt API hit limits.
    5. The UI for the app is made using html. Place your html files in a folder called “templates” for Flask to be able to detect it.
    6. “home.html” is the main file which generates the UI. “layout .html” is a template for other html files so that you don’t need to copy paste the same code multiple times. In this case there’s just one page but making a template is a good practice in general.
    7. Place your css files in a folder called “static” for Flask to be able to detect it.
    8. In the end, your directory should look like this
    9. To run your program, you can double click “app.py” or run it through CMD.
